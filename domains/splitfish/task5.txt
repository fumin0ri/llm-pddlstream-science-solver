For this problem there are six pools connected in a circle, pool 1 is connected both to pool 2 and pool 6 and so on. There is also an additional connection between pool 1 and pool 4. Pool 1 contains nice koi. Pool 2 contains bad piranhas. Pool 3 contains bad lionfish. Pool 4 contains nice clownfish. Pool 5 contains more piranhas. Pool 6 contains more lionfish. The goal is to move the koi to pool 3 and the clownfish to pool 6.
